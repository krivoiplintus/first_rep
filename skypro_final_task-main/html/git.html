Git

  • git -v - эта команда выведет версию Git.
  • git init - инициализация (initialisation) репозитория.
  • git clone - клонировать удаленный репозиторий с GitHub себе на компьютер.
  • git status - просмотр изменений, произошедьших в репозитории.
  • git add - добавление файла в отслеживаемые.
  • git commit -m - создание коммита.
  • git push - отправка изменений на удаленный репозиторий.
  • git log - выведет список всех коммитов.
